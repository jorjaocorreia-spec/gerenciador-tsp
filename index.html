Configurar Google Calendar

Como obter as credenciais? (passo a passo)
1

Criar projeto no Google Cloud

Acesse o Google Cloud Console → clique em "Novo Projeto" → dê um nome (ex: "TSP Manager") → clique em Criar.

2

Ativar a Google Calendar API

No menu lateral → APIs e Serviços → Biblioteca → pesquise "Google Calendar API" → clique nela → Ativar.

3

Configurar a tela de consentimento OAuth

APIs e Serviços → Tela de consentimento OAuth → Tipo: Externo → preencha o nome do app → adicione seu e-mail como usuário de teste → Salvar.

4

Criar a API Key

Credenciais → + Criar Credencial → Chave de API. Após criar, clique na chave → em Restrições de API selecione "Restringir chave" → escolha Google Calendar API → Salvar. Copie a chave gerada.

5

Criar o OAuth 2.0 Client ID

Credenciais → + Criar Credencial → ID do cliente OAuth → Tipo: Aplicativo da Web. Em "Origens JavaScript autorizadas", adicione a URL desta plataforma. Clique em Criar e copie o Client ID.

6

Inserir as credenciais abaixo

Cole o Client ID e a API Key nos campos abaixo e clique em Salvar Configurações.

Como funciona a integração com o Google Calendar

1

Conecte sua conta Google (primeira vez)

Clique no botão Sincronizar Google no topo da tela. Uma janela do Google será aberta pedindo permissão. Autorize e pronto — a conexão fica ativa durante a sessão.

2

Criar ou editar eventos na plataforma

Ao criar ou editar um agendamento, o campo Sincronizar com Google Calendar já vem marcado automaticamente. Salve normalmente — o evento vai direto para o Google sem nenhuma ação extra.

3

Sincronizar o que está no Google Calendar

O botão Sincronizar Google faz os dois sentidos: traz eventos do Google para cá e envia para o Google eventos que ainda não foram sincronizados. Use-o sempre que quiser atualizar tudo de uma vez.

A conexão se renova a cada sessão

O Google exige reautorização a cada vez que você abre a plataforma. Se perceber que um evento não foi para o Google, clique em Sincronizar Google para reconectar e reenviar tudo.

Configurações de credenciais

O ícone de engrenagem ao lado do botão de sincronização permite configurar o Client ID e API Key do Google Cloud, caso necessário.
